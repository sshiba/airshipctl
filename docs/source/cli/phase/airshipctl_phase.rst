airshipctl phase
----------------

Airshipctl command to manage phases

Synopsis
~~~~~~~~


Provides capabilities for interacting with phases, such as getting list of phases or applying a specific one.


Options
~~~~~~~

::

  -h, --help   help for phase

      --debug                enable verbose output
